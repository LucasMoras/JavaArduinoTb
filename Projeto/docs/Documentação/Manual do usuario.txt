Para utilizar o nosso programa, basta clonar o repositório para sua pasta desejada, em seguida utilizando um programa como VsCode para abrir o código, em seguida baixe a IDE do Arduino e implemente o código no mesmo, coloque para rodar após ter inserido a placa na porta serial, em seguida rode o programa Java e selecione a porta onde seu Arduino esta conectada, que será descrita como COM(algum número), clique em conectar e verifique se a mensagem de confirmação apareceu, envie a mensagem desejada e clique enviar, ele deve responder com a mensagem enviada. Note que o programa java de ser aberto pós a abertura e conexão do Arduino, para que reconheça adequadamente.
